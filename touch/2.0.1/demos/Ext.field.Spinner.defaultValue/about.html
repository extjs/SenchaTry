Setting the default value for an Ext.field.Spinner in Sencha Touch 2

The following example shows how you can set the default value on an Ext.field.Spinner (spinnerfield) component in Sencha Touch 2 when no value is defined by setting the defaultValue config option.

Tags:

  • Sencha Touch
  • Sencha Touch 2.0.1
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.form.Panel', {
            fullscreen: true,
            items: [{
                xtype: 'fieldset',
                instructions: 'value = undefined; defaultValue = 9;',
                items: [{
                    xtype: 'spinnerfield',
                    increment: 1,
                    label: 'defaultValue = 9',
                    value: undefined,
                    defaultValue: 9,
                    minValue: 1,
                    maxValue: 12
                }] // items (fieldset)
            }] // items (formpanel)
        }); // create()
    } // launch
}); // application()
