Determining if an image successfully loaded using the Ext.Img component in Sencha Touch 2

The following example shows how you can determine whether an image was able to be successfully loaded using the Ext.Img component in Sencha Touch 2 by using the listeners config object and listening for the load and/or error events.

Tags:

  • Sencha Touch
  • Sencha Touch 2.0.0
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        var logo = Ext.create('Ext.Img', {
            src: 'http://www.sencha.com/img/sencha-large.png',
            width: 372,
            height: 140,
            listeners: {
                load: function (img, evt) {
                    Ext.Msg.alert('Ext.Img loaded successfully');
                }, // load
                error: function (img, evt) {
                    Ext.Msg.alert('Ext.Img could not be loaded', img.getSrc());
                } // error
            } // listeners
        }); // create()
        
        Ext.create('Ext.Panel', {
            fullscreen: true,
            items: logo
        }); // create()
    } // launch
}); // application()
