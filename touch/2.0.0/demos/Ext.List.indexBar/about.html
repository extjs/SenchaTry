Displaying an index bar on an Ext.List component in Sencha Touch 2

The following example shows how you can add an index bar on an Ext.List component in Sencha Touch 2 by setting the Boolean indexBar config option to true.

Tags: Sencha Touch Sencha Touch 2.0.0
[example]
© 2012 Sencha Inc. 
/*global Ext:false */
Ext.application({
    launch: function () {
        Ext.create('Ext.List', {
            fullscreen: true,
            indexBar: true,
            grouped: true,
            store: {
                fields: ['name', 'league', 'division'],
                sorters: 'name',
                grouper: {
                    groupFn: function (item) {
                        return item.get('name')[0];
